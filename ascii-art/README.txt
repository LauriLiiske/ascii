Ascii-art is a program which consists in receiving a string as an argument and outputting the string in a graphic representation using ASCII.

Main program and necessary functions can be found in main.go file.

HOW TO RUN THE CODE:
1. Add executable permission for test.sh script by typing "chmod +x test.sh" or "chmod +rwx test.sh" (without "").

2. Run the script with "./test.sh" command. This will print audit tests on the screen.
